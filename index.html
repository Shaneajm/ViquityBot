Viquity Bot

An AI creation by Shane Alexander written in Discord.js

Why Make A Discord Bot?

I chose to make a Discord bot because the current offerings weren't providing to my needs, I also took it on as a challenge to learn Javascript and Discord.js as they seemed like an easy language and dependency to learn.

Visit the Discord.js website

Bot Features

As of now, the bot's features are very minimal, it includes an 8-Ball, ping time, a profile picture viewer, and announcements for roles when they are created or deleted. I've currently been implementing administration features for me that include setting the bot's status and what game it's playing. These are limited to my use obviously but it will make things much easier down the road.
